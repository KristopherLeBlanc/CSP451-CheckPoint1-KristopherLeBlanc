# CSP451 CheckPoint 1 – Static Site Project

This repository contains the static website and supporting files created for CheckPoint 1 in **CSP451 – DevOps and Cloud-Adjacent Topics**. The project demonstrates foundational Git and GitHub workflows, including repository initialization, branching, Conventional Commits, pull requests, and Markdown documentation.

## Description
This project is a simple static site built to practice version control concepts. It includes an HTML landing page, a stylesheet, and a JavaScript file. The repository showcases:

- Local Git setup and commit history  
- GitHub repository creation and configuration  
- Feature-branch workflow with pull requests  
- Markdown documentation and project structure  

Technologies used: **HTML5**, **CSS3**, **JavaScript**, **Git**, **GitHub**, **GitHub CLI**.
